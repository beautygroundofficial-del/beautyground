영상은 화면 전체를, 채팅·구매는 그 위에 겹치게

지금은 영상 박스 아래로 상품·채팅이 순서대로 나열돼 있어서 스크롤해야 다 보이고, 틱톡·유튜브 라이브 같은 몰입감이 없습니다. 아래 시안은 영상을 화면 꽉 채우는 배경으로 쓰고 그 위에 채팅·좋아요·판매 상품을 오버레이로 얹는 방식입니다. 라이브 커머스 특성상 시청 중 바로 구매까지 이어져야 해서, 채팅으로 화면을 완전히 덮지 않고 상품은 항상 붙박이로 보이게 했습니다.

영상 — 박스 → 전체화면

화면 위쪽 좁은 박스 → 화면 전체가 영상. 세로 방송이면 꽉 차게, 가로 방송이면 위아래 여백만.

채팅 — 아래 별도 칸 → 영상 위 오버레이

스크롤해야 보이던 채팅창 → 영상 하단에 최근 메시지가 반투명하게 떠서 스크롤 없이 바로 보임. 닉네임 탭하면 멘션.

좋아요 하트 — 위치 조정

영상 모서리에 어색하게 떠있던 위치 → 우측 아이콘 레일로 정리, 카운트 표시 추가.

판매 상품 — 스크롤 없이 항상 노출

지금 판매 중인 상품 1개를 채팅 위에 작은 카드로 고정 — 탭하면 구매 시트가 올라오는 방식(전체 목록은 우측 쇼핑백 아이콘).

입력창 — 이모지 버튼 추가

채팅 입력창 왼쪽에 이모지 버튼. 표준 유니코드 이모지라 저작권 문제 없음.

참고 — 이 화면은 영상 위에 얹는 오버레이 UI라 라이트/다크 모드 구분 없이 항상 어두운 톤으로 고정했습니다 (틱톡·인스타 라이브도 동일한 방식). 페이지의 다른 화면들은 기존처럼 라이트/다크 모두 대응됩니다.
