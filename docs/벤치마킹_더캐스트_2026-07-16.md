## 기능 구성 (Play 스토어 설명 기준)
1. 라이브 방송 + 즉시 구매 (고화질, 백화점 내 스튜디오 송출)
2. **라이브 중에만 열리는 특별 혜택** — 방송 시간에만 공개되는 단독 구성·할인
3. 영상 콘텐츠형 제품 정보 (긴 설명 대신 영상)
4. 피드/쇼츠 — 라이브 놓쳐도 짧은 영상으로 구매 연결
5. 전문가 큐레이션 브랜드 셀렉션 (편집샵 정체성)
6. 다시보기 (기한 내 재구매 가능)

## 전략적 시사점 (임원 제안 관점) ⭐
1. **모델 검증**: 롯데가 "백화점 내 라이브 스튜디오+편집샵" 모델을 이미 실행 중 → 우리 제안이 검증된 트렌드라는 근거. 단, **롯데 제안 시 더캐스트와의 차별화 질문이 반드시 나옴** — 답 준비 필수.
2. **AK가 빈자리**: AK에는 이런 파트너가 없음 → 우리의 AK 2개점 운영 실적 + 이 앱이면 최우선 타깃은 AK.
3. **격차가 크지 않음**: 더캐스트도 다운로드 1천+의 초기 단계. 우리가 늦지 않았다.

## 앱 UI 상세 분석 (2026-07-16, 대표님 캡처 18장 — 다운로드\앱사진)

**정보 구조 (하단 탭 6개)**: 홈 / 카테고리 / 브랜드 / 피드 / 좋아요 / 프로필

1. **홈 = 영상 우선**: 최상단이 배너가 아니라 **REPLAY 영상 캐러셀(자동재생·음소거 토글)**. 그 아래 퀵메뉴(브랜드·전체상품·LIVE·캐스트 에디터·다시보기·라이브예고) → **라이브 예고 날짜별 캘린더(오늘~+5일 탭)** → 다시보기 → BEST → 추천. 홈에 들어오면 무조건 영상부터 보게 되는 설계.
2. **방송 네이밍**: "더캐스트 x 라뮤즈"처럼 전부 **브랜드 콜라보 형식** — 방송 자체가 브랜드 제휴 상품(우리 파트너 모델과 동일 문법).
3. **카테고리 화면 상단에 라이브 배너 상시 노출** — 어느 화면에 있든 라이브로 유입시키는 장치.
4. **카테고리 IA**: 좌측 대분류 사이드바(스킨케어~디바이스·향수·푸드·패션·홈) + 우측 소분류 2단 구조. 뷰티 중심 + 라이프스타일 확장(수면·홈프래그런스·푸드) = 백화점 편집샵 그대로.
5. **브랜드 탭**: 브랜드 로고+슬로건+1·2·3 랭킹 상품 캐러셀 — 브랜드 스토리 중심 진열 (84만원 디바이스 등 고가품 포함).
6. **피드 탭**: 틱톡식 세로 쇼츠 + 팔로우. 자체 계정 콘텐츠로 운영 중.
7. **프로필**: **카카오 소셜 로그인**. 최상단에 **"더캐스트 X 롯데백화점 — 매장 연동" 배너**(청량리점 혜택) ← 오프라인 고객을 앱 회원으로 묶는 핵심 장치. 적립금·쿠폰·주문목록·팔로우.
8. **허점(그들도 초기)**: 라이브 예고 캘린더가 "예정된 방송 없음", 방송 빈도 낮아 보임(다시보기 7/15·7/16 2건). 콘텐츠 공급이 아직 얇음.
