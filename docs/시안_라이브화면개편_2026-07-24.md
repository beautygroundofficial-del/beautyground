# 라이브 시청 화면 개편 시안 (2026-07-24)

**배경**: 라이브 송출 실검증(RTMPS→Cloudflare→재생) 중, 실제 화면을 보고 대표님이 "틱톡/유튜브 라이브에 비해 어색하다" 지적. 채팅·이모지·멘션·좋아요 위치가 다 아래로 나열돼있어 몰입감이 없고, 전문 라이브커머스로서 이 정도는 안 된다는 판단.

**시안**: `시안_라이브화면개편_2026-07-24.html` (같은 폴더). Artifact로도 공개: https://claude.ai/code/artifact/c8565bbf-147a-4f97-beaa-9bcca404881c

**대표님 승인 완료(2026-07-24)** → `ShopLiveWatch.tsx` 실제 코드에 반영 진행.

## 변경 요지
1. 영상 — 상단 박스 → 화면 전체(풀블리드)
2. 채팅 — 아래 별도 카드 → 영상 하단 반투명 오버레이 (스크롤 없이 항상 보임), 닉네임 탭 = 멘션(@닉네임)
3. 좋아요 하트 — 어색한 모서리 위치 → 우측 아이콘 레일로 정리 + 카운트
4. 판매 상품 — 아래 나열 → "지금판매" 1건 항상 노출되는 미니카드(탭하면 구매 시트) + 우측 레일 쇼핑백 아이콘(전체 목록)
5. 채팅 입력창에 이모지 버튼 추가 (표준 유니코드, 저작권 문제 없음)

## 참고 벤치마크 (커머스 특화라 틱톡/유튜브보다 더 정확한 참고)
- 그립(GRIP) — 상품 트레이 항상 하단 고정
- 카카오쇼핑라이브
- 네이버 쇼핑라이브 — 채팅+상품+영상 배치가 이 시안이랑 제일 비슷함
