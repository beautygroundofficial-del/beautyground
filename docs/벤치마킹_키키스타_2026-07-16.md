# 벤치마킹 — 키키스타 (2026-07-16)

> 출처: Google Play "KiKi 키키방송" 앱 정보·스크린샷·리뷰 130여 개 (대표님 수집)

## 서비스 개요
소상공인·개인용 "1인 방송국" 앱. 라이브커머스는 여러 기능 중 하나이고 본질은 소셜 방송 플랫폼
(선물·포인트 수익, 추천 보상, 노래방, 화상채팅, 지도 방송 등). 다운로드 5만+, 평점 4.3.

## 배울 것 (리뷰에서 실제로 칭찬받은 강점)
1. **진입장벽 제로** — "장비 없이 폰 하나로 방송"이 최다 언급 칭찬. 방송 시작이 쉬울수록 판매자가 늘어남
2. **양방향 소통** — 시청자가 참여한다는 감각 (채팅·게스트·리액션)
3. **쿠폰방송** — 방송 중에만 쓰는 혜택으로 시청→구매 전환 유도
4. **판매자 채널화** — 마이홈에 외부 링크, 방송 종료 후에도 유입 지속

## 반면교사 (버릴 것)
- 2024년 이후 리뷰 급변: 탈퇴 불가·결제취소 불가·CS 불통·접속 불가 → **기능 백화점식 확장으로 기본(결제·CS·안정성)이 무너진 사례**
- 포인트·선물·추천수익 = 다단계성 논란·정산 분쟁 위험. 커머스인 우리에겐 불필요
- 노래방·화상통화·지도방송 = 뷰티 브랜드 라이브 판매 방향과 무관

## 우리 로드맵 반영 (우선순위)
| 우선순위 | 항목 | 비고 |
|---|---|---|
| 지금 (기존 로드맵) | ② 지금판매·공지핀 실시간 동기화, ③ 다시보기 | 강점 1·2 커버 |
| 다음 후보 | 라이브 한정 쿠폰/특가 (방송 중에만 그 가격) | 쿠폰방송 벤치마킹, 전환율 직결 |
| 다음 후보 | 방송 예약 알림 ("알림받기") | 시청자 모객 핵심 |
| 나중 | 브라우저 바로 송출 (Cloudflare WHIP, Larix 불필요) | "장비 제로" 완성 |
| 나중 | 다시보기 → 쇼트클립 하이라이트 | 방송 후 유입 자산화 |
| 안 함 | 포인트·선물, 노래방, 화상채팅, 지도, 추천수익 | 방향 불일치 + 복잡성 |

**결론**: 가져올 건 기능 목록이 아니라 두 원칙 — "판매자는 폰 하나로, 시청자는 방송 중 혜택으로".

## 웹 직접 조사 추가 (2026-07-16, 공식 사이트·기업정보)
- **회사**: 키키스타 주식회사 (㈜키키스타방송), 대표 강은숙, 본사 경기 포천. **연매출 약 10억** (잡코리아 기업정보)
- **현재 상태**: 사이트(kikistar.co.kr) 정상 운영 중 — 2024년 "접속 불가" 리뷰들에도 서비스는 존속
- **포지셔닝 전환 확인**: 현재 랜딩은 소셜방송이 아니라 **"라이브커머스 KiKi-STAR"**를 전면에 — 4대 서비스: LIVE(라이브방송) / **INSTANT(라이브 특가)** / PLEASURE(방송 후 VOD) / PREMIUM(차별화 상품·이벤트)
- **동네방송국 전략**: 전국 250개 지자체에 "동네방송국" 개국 추진 (지역 франчайз형 확장) — 우리와 무관하나 참고
- **수익모델 주의**: 상단 메뉴에 "정기결제"(구독) — 리뷰의 결제취소 분쟁 원인으로 추정. 우리는 구독모델 지양 근거
- **시사점**: 키키스타(INSTANT)·더캐스트(라이브 단독혜택) 모두 **"라이브 한정 특가"를 핵심 소구점**으로 잡음 — 경쟁 2사 공통 → 우리 기능 1순위 근거 강화. 반면 이들과 달리 우리는 백화점 오프라인 실적이라는 신뢰 앵커 보유(키키스타에 없는 것)
