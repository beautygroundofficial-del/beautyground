# 뷰티그라운드 몰 — UI 디자인 요청서 (2026-07-16)

## 1. 프로젝트 개요
- **서비스**: 뷰티그라운드 라이브커머스 쇼핑몰 (모바일 웹앱)
- **현재 사이트**: https://beautyground.vercel.app/app/home (참고용 — 현재 화면 그대로 확인 가능)
- **배경**: 백화점(AK플라자 광명·수원) 오프라인 매장 기반의 뷰티 편집샵이 라이브 방송으로 판매하는 서비스. 백화점 임원 대상 시연이 예정되어 있어 **프리미엄 완성도**가 중요.

## 2. 작업 범위 — 핵심 6화면 (모바일 기준)
1. **홈** — 라이브/다시보기 영상이 상단에 오는 영상 우선 구조 희망
2. **라이브 시청** — 세로 영상 + 판매상품 카드 + "지금 판매 중" 하이라이트 배지 + 공지핀 + 실시간 채팅 (기능 요소는 현재와 동일, 배치·스타일만 재설계)
3. **라이브 목록** — 방송 예정(날짜별)·진행중·다시보기
4. **상품 상세**
5. **주문/결제** — 배송지 입력, 결제 금액, 결제 버튼 (흐름 변경 불가, 스킨만)
6. **마이페이지** — 상단에 "뷰티그라운드 X AK플라자 매장 연동" 배너 영역 포함

+ 공통: 하단 탭바, 상단 헤더, 상품 카드 컴포넌트

## 3. 납품 형식 (중요)
- **Figma 파일/링크** (코드·퍼블리싱 불필요 — 구현은 자체 개발로 진행)
- 화면 시안 + **디자인 토큰 정리**: 컬러 팔레트, 폰트(크기·굵기 단계), 여백 규칙, 버튼/카드 컴포넌트
- 기준 화면 폭: **모바일 480px** (PC에서는 480px 프레임이 화면 중앙에 고정되는 구조 — 시안은 모바일만 주시면 됩니다)
- 아이콘: 이모지 대신 라인 아이콘 세트 제안 요청

## 4. 톤 & 레퍼런스
- 지향점: **백화점 편집샵의 프리미엄** — 화려함보다 정제됨
- 레퍼런스: 더캐스트 앱(다크 테마·영상 우선·브랜드 큐레이션), med-ligne.com (현재 홈이 참고한 사이트)
- 현재 브랜드 요소: 크림 배경 + 골드 포인트 (유지할지 새 방향 제안할지는 디자이너 제안 환영)
- 실제 콘텐츠(브랜드 상품 사진·방송 썸네일)가 들어간다는 전제로 시안 구성 요청 (Lorem ipsum성 회색 박스 지양)

## 5. 참고 자료 제공 가능
- 현재 서비스 전 화면 캡처
- 경쟁사(더캐스트) 화면 캡처 18장
- 등록된 실제 상품 이미지 (쁘띠페 등 70여 개)

## 6. 일정·소통
- 1차 시안 → 피드백 1회 → 최종본 (수정 범위 사전 협의)
- 시안 확정 후 개발 반영은 자체적으로 2~3일 내 완료 예정
