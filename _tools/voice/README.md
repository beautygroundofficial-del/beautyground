# F9 음성 입력 (음성자동전송.ahk)

F9 한 번 → 윈도우 음성 받아쓰기 시작 → 말하기 → F9 다시 → 자동 전송(Enter). Esc는 취소.
Claude Code 등 아무 입력창에서나 동작. 내부적으로 Win+H(윈도우 음성 입력)를 대신 눌러주는 방식.

## 새 컴퓨터(사무실)에 설치하는 법

1. **AutoHotkey v2 설치**: https://www.autohotkey.com/ → Download → v2.0 설치 (기본 옵션 그대로)
2. 이 폴더의 `음성자동전송.ahk` **더블클릭** → 즉시 F9 사용 가능
3. **부팅 시 자동 실행 등록**(안 하면 재부팅마다 2번을 다시 해야 함):
   - `Win+R` → `shell:startup` 입력 → 엔터 → 열린 폴더에 `음성자동전송.ahk`의 **바로가기**를 넣기
     (파일 우클릭 → 보내기 → 바탕화면 바로가기 만들기 → 그 바로가기를 시작프로그램 폴더로 이동)

## 주의

- 한국어 인식은 키보드가 **한글 상태**여야 함
- 말 끝나고 F9를 누르면 약 2초 후 전송됨(마지막 문장 인식 확정 대기 시간 — 정상)
- F9가 안 먹으면: 이 파일을 다시 더블클릭 (트레이에 초록 H 아이콘이 떠 있는지 확인)
